Hospede o HTML no GitHub Pages ou Netlify e cole a URL aqui. Ela será usada nos links de convite do Match.

1. Crie conta em github.com
2. Crie repositório público → faça upload do arquivo HTML

Ou use netlify.com → arraste o arquivo HTML → copie a URL.
